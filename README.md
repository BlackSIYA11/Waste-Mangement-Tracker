# Waste-Mangement-Tracker

**Overview**
The Waste Management Tracker is a WPF (Windows Presentation Foundation) application designed to help manage user registrations and logins for a waste management system. The application allows users to register with their details, log in to access the main system, and manage their activities. It interacts with a database for storing and retrieving user information, including usernames, email addresses, and passwords.

**Features**
User Registration: Allows users to create an account with a username, email, and password.
User Login: Provides functionality for registered users to log in.
Database Integration: Data is stored and fetched from a database, ensuring persistent user data.
Password Security: Passwords are securely hashed before storing in the database.
User-friendly Interface: WPF-based interface with placeholder text, user input validation, and visual feedback.
Technologies Used
C#: Main programming language used for the application.
WPF: Windows Presentation Foundation for building the user interface.
SQL Database: A database for storing user information (using a SQL Server.
BCrypt: Password hashing library for secure password management.
XAML: Markup language for designing the application's user interface.
